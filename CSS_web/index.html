天界學園放送部

天音彼方（日語：天音かなた／あまね かなた Amane Kanata */?、4月22日[2] - ）是日本的虛擬YouTuber、虛擬偶像。 所屬事務所為Hololive Production[2]，為Hololive四期生的成員[3]。 又被粉絲稱為「PP（PowerPoint）天使」[3][4][5]、大猩猩。[4][5]與同為四期生的桐生可可同居。

簡介

角色設定

天界学園に通う天使。人を癒すための研究をしている。 恥ずかしがりやな性格を隠すために、クールを装おうとする。 天界学园上学的天使。正进行着治愈人类的研究。 为了掩盖自己怕羞的性格、装成很酷的样子。

活動歷程

    2019 :

  1. 12月25日，hololive宣布四期生的出道，分别为天音彼方、桐生可可、角卷绵芽、常暗永远、姬森璐娜。同日推特和YouTube频道开始公开[2]。 按照告知，天音彼方、桐生可可、角卷绵芽三人于12月25日当日按照顺序开始活动。常暗永远、姬森璐娜则在2020年1月1日方才开始活动。 同日天音彼方推特开始活动，发表第一条推文[3]。
  2. 12月27日，在YouTube平台进行初直播

    2020 :

  • 2月8日，YouTube频道订阅数达到10万
  • 2月9日，YouTube频道收益通过
  • 3月29日因耳疾复发住院[11]，3月31日出院，4月1日发布复活直播[12]。
